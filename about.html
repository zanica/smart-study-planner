About This Website

The Gamified Smart Study Planner is a client-side web application designed to support students in organising their academic workload while making the process more engaging and enjoyable. Traditional task management tools often focus purely on functionality, which can lead to reduced motivation over time. In response to this, this application integrates core planning features—such as task creation, editing, prioritisation, and deadline tracking—with gamification elements that encourage consistent user interaction. Users can earn points for completing tasks, progress through different levels, and maintain daily streaks, all of which are designed to reinforce positive study habits. Visual and audio feedback, including animations and sound effects, further enhance the user experience by providing immediate and rewarding responses to user actions. The system is built using HTML, CSS, and JavaScript, and utilises browser-based storage to persist data without requiring a backend. Overall, the application aims to demonstrate how interactive design and gamification techniques can transform a simple productivity tool into a more motivating and user-friendly experience.
